data ConstantTy Source #

Enumerates all supported types of constants. This would technically be a great candidate for a data family, but we'd then end up with some mutually recursive modules, unless we shuffle things around a bunch.

type MonadCodeGen m = (MonadIO m, MonadFail m, MonadModuleBuilder m, MonadReader CodeGenSt m, MonadFix m) Source #

A constraint synonym for the type class constraints we usually expect our code generation monad to satisfy.

mkFresh :: (MonadReader CodeGenSt m, MonadIO m) => String -> m String Source #

mkFresh prefix generates a fresh name starting with prefix.

extendScope :: MonadReader CodeGenSt m => Text -> Local -> m a -> m a Source #

extendScope name operand action adds name to the scope for action along with operand which represents the LLVM identifier.

updateEnv :: MonadReader CodeGenSt m => Set Text -> [Local] -> m a -> m a Source #

updateEnv names operands action updates the scope for action with operands which correspond to names. Important: the names in names must be in the same order as the corresponding entries in operands.

localFreeVars :: MonadReader CodeGenSt m => Set (Text, Bool) -> m a -> m a Source #

localFreeVars freeVars sets the set of free variables in the current compilation scope to freeVars.

currentFreeVars is a computation which retrieves the set of free variables in the current compilation scope.
