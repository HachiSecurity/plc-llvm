Description

This module provides convenience functions for constructing LLVM IR, including more flexible versions of functions found in IRBuilder.

global name type initialValue transformation emits a new global named name of type with an initial value of initialValue. Additionally, transformation is applied to the global before it is emitted, so that arbitrary transformations can be performed on it.
